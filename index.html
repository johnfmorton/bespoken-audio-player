Bespoken Audio Player Demo Page

The `bespoken-audio-player` element is a custom element that allows you to play audio files in a playlist. The element is built using the Web Components API and is compatible with all modern browsers.

In this first example, the `bespoken-audio-player` element is used to play a single audio file. 

          tracks='[{"src": "demo-page-assets/hello-america-audio-20240913-063213.mp3"}]'

The `only-current-track-visible` attribute will override the `playlist-visible` attribute and only show the current track. 

  <bespoken-audio-player
    playlist-visible
    tracks='[
        {"src": "demo-page-assets/fish-and-chips.mp3", "title": "Introduction"},
        {"src": "demo-page-assets/hello-america-audio-20240913-063213.mp3"},
        {"src": "demo-page-assets/hello-britain-audio-20240913-063749.mp3", "title": "Conclusion"}
      ]'>
  </bespoken-audio-player>
