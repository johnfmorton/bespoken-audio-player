Demo: Bespoken Audio Player

The `bespoken-audio-player` element is a custom element that allows you to play audio files in a playlist. The element is built using the Web Components API and is compatible with all modern browsers.

Example 1: Single track

In this first example, the `bespoken-audio-player` element is used to play a single audio file. 

<bespoken-audio-player
    class="mb-6 border border-gray-300 rounded bg-slate-100 block p-5"
    tracks='[{"src": "hello-america-audio-20240913-063213.mp3"}]'
    >
</bespoken-audio-player>

Example 2:

The `only-current-track-visible` attribute will override the `playlist-visible` attribute and only show the current track. Notice that there are some Tailwind CSS classes added to the element to style the overall component. 

<bespoken-audio-player class="mb-6 border border-gray-300 rounded bg-slate-100 block p-5"
    only-current-track-visible
    tracks='[
      {"src": "fish-and-chips.mp3", "title": "Introduction"},
      {"src": "hello-america-audio-20240913-063213.mp3"},
      {"src": "hello-britain-audio-20240913-063749.mp3", "title": "Conclusion"}
    ]'
    >
</bespoken-audio-player>
